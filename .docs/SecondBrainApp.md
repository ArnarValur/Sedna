# Second Brain App

_*The Problem*_: In Google Keep Notes I have a note that has bunch of URLs of interesting article and things that I find on my phone, some of these Ive thought to "read later" but now its a long list of all kinds of URLs that Ive forgotten.

_*Solution*_: We do the same what Obsidian Clipper does, retrives the Page Variables like content, title, url, and description (if there is one) creates a markdown file with those variables and syncs it to a Shared Drive on Google Drive org account. There is where I store my Obsidian Vault. So if we create an app that performs the same functionality like Obsidian Clipper and outputs a markdown file into our shared drive,

It would be cool if this app can recieve a shared link from Android, and from there it does the rest to fetch the content and export the markdown file up to our shared drive.

This is a practice project, presumably we need OAuth and whatnot.
We have full Firebase and GCP to our disposal.
