# Tech Stack

- **Frontend**: Flutter (Mobile & Web/Desktop). Language: Dart.
- **Sharing**: `receive_sharing_intent` package to handle incoming links.
- **AI**: `google_generative_ai` package (Dart SDK for Gemini).
- **Backend**: Firebase (Auth, Firestore). All Firebase Functions deploy to `europe-west1`.
- **Scraping**: `http` and `html` packages (to extract text from URLs).
